# nodejs-api
NodejS + server + rest api

Follow this steps to up an node server with an default API

## Setup

### 1
```
npm init -y
```

### 2

```
npm install express
```

### 3
create `index.js` and copy content from this template.

## Run

```
node src/index.js
```

## Test

Once server is running call this URL:
```
http://localhost:3000
```
